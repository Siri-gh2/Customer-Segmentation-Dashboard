# Customer Segmentation Dashboard

## Overview
This project is an interactive Customer Segmentation Dashboard developed using Power BI to analyze customer behavior, sales contribution, profitability, and regional customer distribution. The dashboard transforms raw sales data into meaningful customer insights using data visualization and business intelligence techniques.

The objective of this project is to identify valuable customer segments, analyze purchasing patterns, and support data-driven business decisions through interactive analytics.

---

## Features

- KPI Cards for:
  - Total Customers
  - Total Sales
  - Total Profit
  - Average Sales Per Customer

- Segment-wise Sales Analysis
- Segment Profit Distribution
- Region-wise Customer Analysis
- Top 10 Customers by Sales
- Interactive Slicers for:
  - Segment
  - Region
  - Category

---

## Tools & Technologies Used

- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Excel Dataset

---

## Dataset

The dashboard uses the Superstore Sales Dataset containing:
- Customer details
- Product categories
- Regional sales data
- Profit information
- Customer segments
- Order details

---

## Key Insights

- Consumer segment generated the highest sales contribution.
- Corporate customers showed strong profitability.
- West region had the highest customer concentration.
- Certain customers generated high revenue with lower profit margins.

---

## Dashboard Components

### KPI Section
Displays key customer and business performance metrics.

### Segment-wise Sales Analysis
Compares sales performance across customer segments.

### Profit Distribution
Visualizes profit contribution by customer segment.

### Region-wise Customer Analysis
Shows customer distribution across regions.

### Top Customers Analysis
Highlights top-performing customers based on sales and profit.
